核心代码片段：

.cube {
    transform-style: preserve-3d;
    transform: translateZ(-150px) rotateX(-15deg) rotateY(15deg);
}

.front {
    background: linear-gradient(135deg, #ff416c, #ff4b2b);
    transform: translateZ(100px);
}

.right {
    background: linear-gradient(135deg, #2193b0, #6dd5ed);
    transform: rotateY(90deg) translateZ(100px);
}

/* 倾斜变换示例 */
.skewed {
    transform: translateZ(-150px) skewX(10deg) skewY(-5deg);
}
这个3D立方体使用CSS的transform属性创建，每个面都应用了linear-gradient背景。 使用了translateZ、rotateX、rotateY、skewX和skewY等3D变换。
